## Autenticación con credenciales

Este endpoint permite generar tokens JWT para acceder a los recursos protegidos de AgroSoft API.
el cual debera estar en los Headers de las solicitudes

## Headers

| Parámetro         | Tipo    | Descripción                                | Requerido |
|------------------|--------|--------------------------------------------|-----------|
| `access_token`   | string | Token de acceso generado previamente      | ✅         |
| `Content-Type`   | string | Tipo de contenido: `application/json`     | ✅         |
| `Accept`        | string | Tipo de respuesta: `application/json`      | ✅         |


### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`correoElectronico`| string  | Sí        | Correo registrado en el sistema | "user@agrosoft.com" |
| `password`         | string  | Sí        | Contraseña del usuario          | "PassSecure123" |


## Respuestas

✅ 201 - Éxito
```json

{
  "access": "eyJhbGciOiJIUz...",
  "refresh": "eyJhbGciOiJIUz..."
}
```

❌ 400 - Error
```json
Copy
{
  "detail": "Credenciales inválidas"
}
```
