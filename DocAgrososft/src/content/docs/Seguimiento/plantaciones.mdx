## Creacion de plantaciones

Este endpoint permite crear nuevos plantaciones en AgroSoft Api 

### Campos requeridos

```json
    {
        "fk_Cultivo": 1,
        "fk_Era": 2
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`fk_Cultivo`          | number    | Sí        | cultivo al que pertenece | 38 |
| `fk_Era`       | number    | Sí        | era a la q pertenece        | 2 |


## Respuestas Postplantaciones

```json
    {
```

Este endpoint permite editar las plantaciones existentes en AgroSoft Api

```json
        "id": 1,
        "fk_Era": 2
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`id`          | number    | Sí        | id dla plantacion | 38 | 1
| `fk_Era`     | number    | Sí        | era a la que pertenece        | 3 |
