## Creacion de eras

Este endpoint permite crear nuevos eras en AgroSoft Api 

### Campos requeridos

```json
    {
        "fk_lote": 1,
        "tamX": 1,
        "tamY": 2,
        "posX": "3.00",
        "posY": "4.00",
        "tipo": disponible,
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`fk_lote`          | number    | Sí        | lote al que pertenece| 1|
| `tamX`     | number    | Sí        | tamaño en X        | 1 |
| `tamY`    | number    | Sí        | tamaño en Y       | 1 |
| `posX`       | number    | Sí        | posicion en X       | 1 |
| `posY`       | number    | Sí        | posicion en Y        | 1 |
| `tipo`       | string    | Sí        | estado de la era       | disponible |
