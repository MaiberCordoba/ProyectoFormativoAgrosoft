## Creacion de Semilleros

Este endpoint permite crear nuevos semilleros en AgroSoft Api 

### Campos requeridos

```json
    {
        "unidades": 38,
        "fechasiembra": "2025-03-26",
        "fechaestimada": "2025-03-31",
        "fk_especie": 2
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`unidades`          | number    | Sí        | Cantidades de semillas | 38 |
| `fechasiembra`     | date    | Sí        | fecha de la siembra         | "2025-03-26" |
| `fechaestimada`    | date    | Sí        | fecha estimada de la cosecha        | "2025-03-31" |
| `fk_especie`       | number    | Sí        | especie a la q pertenece        | 2 |
