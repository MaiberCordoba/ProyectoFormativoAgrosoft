## Obtencion de tipos especie

Este endpoint permite llamar todos los tipos especie existentes en AgroSoft Api 
para traerlos y en listarlos

```json

    {
        "id": 4,

        "descripcion": "es papa",
        "img": "png"
    }

```

```json
Copy

  "detail": "No se encontraron tipos especie"
}
```

## Creacion de tipos especie

Este endpoint permite crear nuevos tipos especie en AgroSoft Api 

### Campos requeridos

```json
    {
        "nombre": "Papa",
        "descripcion": "es papa",
        "img": "string"
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`nombre`          | string    | Sí        | nombre del tipo de especie | Papa |
| `descripcion`     | string    | Sí        | descripcion del tipo de especie         | es papa |
| `img`    | date    | string    |Sí    | imagen del tipo de especie       | imganeXD.png |

## Edicion de tipos especie

Este endpoint permite editar los tipos especie existentes en AgroSoft Api 

### Campos requeridos

```json
    {
        "id": 4,
        "nombre": "Papa",
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`id`          | number    | Sí        | id del tiposespecie |   1
| `nombre`     | string    | Sí        | nombre del tipo de especie         | Yuca |
