## Creacion de lotes

Este endpoint permite crear nuevos lotes en AgroSoft Api 

### Campos requeridos

```json
    {
        "nombre": "lote de papaya",
        "descripcion": "papaya en lote",
        "tamX": 1,
        "tamY": 2,
        "estado": false,
        "posX": "3.00",
        "posY": "4.00"
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`nombre`          | string    | Sí        | nombre del lote | lote de papaya |
| `descripcion`     | string    | Sí        | descripcion del lote         | papaya en lote |
| `tamX`    | number    | Sí        | tamaño en X        | 1 |
| `tamY`       | number    | Sí        | tamaño en Y        | 2 |
| `estado`       | boolean    | Sí        | estado del lote        | true or false |
| `posX`       | number    | Sí        | posX        | 2 |
| `posY`       | number    | Sí        | psoY       | 2 |

## Edicion de lotes

Este endpoint permite editar los lotes existentes en AgroSoft Api 

### Campos requeridos

```json
{
        "id": 1,
        "descripcion": "papaya en lote",
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`id`          | number    | Sí        | id del lote | 1
| `descripcion`     | string    | Sí        | nombre del lote        | papaya en lote dos |
