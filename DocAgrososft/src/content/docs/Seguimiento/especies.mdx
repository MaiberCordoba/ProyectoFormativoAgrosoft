## Creacion de especies

Este endpoint permite crear nuevos especies en AgroSoft Api 

### Campos requeridos

```json
    {
        "nombre": "Mamuel",
        "descripcion": "illo pillo",
        "img": "xd",
        "tiempocrecimiento": 19,
        "fk_tipoespecie": 1
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`nombre`          | string    | Sí        | nombre de la especie | Mamuel|
| `descripcion`     | string   | Sí        | descripcion de la especie        | papita rara |
| `img`    | string    | Sí        | imagen de la especie        | imganeXD.png |
| `tiempocrecimiento`       | number    | Sí        | tiempo de crecimiento en dias       | 19 |
| `fk_tipoespecie`       | number    | Sí        | tipo de especie a la q pertenece        | 1 |


## Respuestas Postespecie

    }
