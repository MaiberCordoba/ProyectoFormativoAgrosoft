## Creacion de cultivo

Este endpoint permite crear nuevos cultivo en AgroSoft Api 

### Campos requeridos

```json
    {
        "nombre": "papa",
        "unidades": 25,
        "activo": true,
        "fechaSiembra": "2025-03-26",
        "fk_Especie": 2
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`nombre`          | string    | Sí        | nombre del cultivo | papa |
| `unidades`     | number    | Sí        | cantidad de cultivo         | 25 |
| `activo`    | boolean    | Sí        |    si esta activo o no    | true or false |
| `fechaSiembra`       | date    | Sí        | fecha estimada de la cosecha        | 2025-03-26 |
| `fk_Especie`       | number    | Sí        | especie a la q pertenece        | 2 |


## Respuestas Postcultivo

## Edicion de cultivo

Este endpoint permite editar los cultivo existentes en AgroSoft Api 

### Campos requeridos

```json
{
        "id": 1,
        "fechasiembra": "2024-03-28",
    }

```
### Parámetros del Request

| Parámetro          | Tipo      | Requerido | Descripción                     | Ejemplo         |
|--------------------|---------- |-----------|---------------------------------|-----------------|
|`id`          | number    | Sí        | id del cultivo | 1
| `fechasiembra`     | "date"    | Sí        | fecha de la siembra         | "2024-03-28" |
