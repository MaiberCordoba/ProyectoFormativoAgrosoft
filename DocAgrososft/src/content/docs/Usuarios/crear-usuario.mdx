## Crear un usuario

<ApiBaseBanner 
  method="POST" 
  path="/usuarios/" 
  baseUrl="http://127.0.0.1:8000"
/>

Este endpoint permite registrar un nuevo usuario en el sistema.

### Campos requeridos
```json
{
  "identificacion": 123456789,
  "nombre": "Juan",
  "apellidos": "Pérez",
  "fechaNacimiento": "1990-01-01",
  "telefono": "123456789",
  "correoElectronico": "juan@example.com",
  "password": "SecurePass123",
  "admin": false
}

```

### Respuesta
✅ 201 - Éxito
```json
{
  "id": 1,
  "nombre": "Juan",
  "correoElectronico": "juan@example.com"
}
```

❌ 400 - Error
```json
{
  "detail": "Datos inválidos o incompletos"
}
```
