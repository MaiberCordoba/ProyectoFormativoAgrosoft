## Actualizar usuario

<ApiBaseBanner 
  method="PUT" 
  path="/usuarios/{id}/" 
  baseUrl="http://127.0.0.1:8000"
/>

Actualiza un usuario por su ID.

### Campos requeridos
```json
{
  "nombre": "Juan Pérez",
  "telefono": "987654321"
}
```

### Respuesta
✅ 200 - Éxito
```json
{
  "id": 1,
  "nombre": "Juan Pérez",
  "telefono": "987654321"
}
```
